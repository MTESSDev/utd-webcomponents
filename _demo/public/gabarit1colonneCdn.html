Gabarit 1 colonne (avec Cdn)

La présente page est une simple page HTML contenant le markup essentiel pour l'utilisation des composants UTD dans un contexte de gabarit 1 colonne en utilisant le cdn.

Inspecter la source de la page afin de voir le HTML requis.

Vous devrez évidemment paramétrer les composants utd de base (utd-piv-entete, utd-piv-pied-page, ainsi que utd-menu-vertical et utd-pied-page-site au besoin. Les pages d'aide de ces composants sont dans le menu à gauche.
