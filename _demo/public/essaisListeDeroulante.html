Exemples liste déroulante

Cette page "statique" est utilisée afin de tester une sélection par défaut dans le contrôle de liste déroulante.

Il semble exister un bug avec notre site démo qui utilise SVELTE. L'attribut "selected" dans les éléments "option" semble retiré par SVELTE, ce qui cause problème pour les essais d'assignation d'une valeur par défaut à une liste déroulante.

1- Liste déroulante à sélection simple avec une option sélectionnée par défaut

L'option sélectionnée par défaut devrait être "Québec".

2- Liste déroulante à sélection simple en lecture seule avec une option sélectionnée par défaut

L'option sélectionnée par défaut devrait être "Québec" et le champ non modifiable.
